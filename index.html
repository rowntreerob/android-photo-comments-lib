android-photo-comments-lib

comment on photos

Commenting on a photo or narrating photos for storytelling are a cinch with "fotocoda" app - as easy as tapping the mic and then talking to the phone and letting the cloud do the rest. Alongside any gallery/image view, tap the mic icon and add a narratative track to the 'current' picture. As soon as the voice-record is stopped, immediately the app pipelines the audio comments and photos to a cloud services platform offering an array of API services and social integration.

Features List

  • comment on any picture with a tap on the microphone icon
  • record and capture voice comment track
  • automatically convert voice comment to text
  • integrate the text version of the audio comment and the picture
  • overlay the text comment on the picture
  • create new mp4 incorporating orig voice comment along with picture
  • media bundle from phone is synched to cloud
  • cloud media layer designed for API integration (HTTP/REST)
  • post to FB, G+, Youtube...
